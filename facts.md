# 상품 정보 입력 예시

이 파일은 상세페이지 제작 전에 상품 정보를 정리하는 예시입니다.
새 상품을 만들 때는 상품 폴더 안에 `facts.md`를 복사해서 사용하세요.

예시 위치:

```text
products/olive-oil/facts.md
```

## 기본 정보

| 항목 | 입력 내용 |
|---|---|
| 상품명 | 올리브오일 |
| 브랜드명 | 확인 필요 |
| 카테고리 | 식품 |
| 판매 채널 | 스마트스토어 / 쿠팡 / 자사몰 / 기타 |
| 상품 한 줄 설명 | 확인 필요 |
